## 자주 묻는 질문 (FAQ)

<AccordionGroup>
  <Accordion title="SDK가 로드되지 않아요.">
    1. 스크립트 URL이 정확한지 확인하세요.
    2. 콘솔 에러를 확인하고 네트워크 연결 상태를 점검하세요.
    3. Content Security Policy (CSP) 설정을 확인하세요.
  </Accordion>

  <Accordion title="상품 정보가 올바르게 전달되지 않아요.">
    1. `cartProducts` 배열의 구조가 올바른지 확인하세요.
    2. 필수 필드가 모두 포함되어 있는지 확인하세요.
    3. 옵션이 없는 상품의 경우, 빈 옵션을 추가했는지 확인하세요.
  </Accordion>

  <Accordion title="결제 과정에서 오류가 발생해요.">
    1. 상품 정보가 올바르게 전달되었는지 다시 한 번 확인하세요.
    2. 네트워크 연결 상태를 점검하세요.
    3. 콘솔에 표시된 오류 메시지를 확인하고, 해당 오류에 대한 해결 방법을 찾아보세요.
  </Accordion>
</AccordionGroup>

## 문제 해결 가이드

### SDK 초기화 문제

<Steps>
  <Step title="스크립트 로드 확인">
    브라우저 개발자 도구의 네트워크 탭에서 SDK 스크립트가 제대로 로드되었는지 확인하세요.
  </Step>
  <Step title="콘솔 에러 확인">
    콘솔에 표시된 에러 메시지를 확인하고, 해당 에러에 대한 해결 방법을 찾아보세요.
  </Step>
  <Step title="데이터 구조 검증">
    `cartProducts` 배열의 구조가 API 문서에 명시된 형식과 일치하는지 확인하세요.
  </Step>
</Steps>
