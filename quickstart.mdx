## 전제 조건

<Check>웹 서버 또는 정적 웹 호스팅 서비스</Check>
<Check>기본적인 HTML, JavaScript 지식</Check>

## 설치 단계

<Steps>
  <Step title="판매자 등록 (선택사항)">
    백엔드 시스템에 판매자 등록을 요청합니다. 이 단계는 선택사항이지만, 등록 시 제공되는 `sellerId`를 사용하면 체크아웃 페이지에 판매자의 로고와 약관 동의 사항이 표시됩니다.
  </Step>
  <Step title="SDK 스크립트 추가">
    HTML 파일의 `<head>` 또는 `<body>` 태그 내에 다음 스크립트를 추가합니다:

    ```html
    <script src="https://sdk-checkout-staging.delivered.co.kr/index.js"></script>
    ```
  </Step>
  <Step title="SDK 사용">
    JavaScript 코드에서 `dkLibrary.order()` 함수를 호출하여 글로벌 오더 프로세스를 시작합니다:

    ```javascript
    const cartProducts = [
      {
        productRequest: {
          title: "샘플 상품",
          price: 10000,
          originUrl: "https://example.com/product",
          images: ["https://example.com/image.jpg"],
          options: [{ title: "", price: 0 }]
        },
        selectedOptions: [{ title: "", quantity: 1 }]
      }
    ];

    dkLibrary.order(cartProducts);
    ```
  </Step>
  <Step title="테스트 실행">
    웹 페이지를 열고 콘솔에 오류가 없는지 확인합니다. 
    결제 버튼을 클릭하여 글로벌 오더 프로세스가 시작되는지 테스트합니다.
  </Step>
</Steps>
