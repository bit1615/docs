## 코어 API

### dkLibrary.order(cartProducts)

글로벌 오더 프로세스를 시작합니다.

<ParamField path="cartProducts" type="CartProductRequest[]" required>
  주문할 상품 정보를 담은 배열
</ParamField>

```javascript
dkLibrary.order(cartProducts);
```

## 데이터 타입

### CartProductRequest

<ResponseField name="productRequest" type="ProductRequest" required>
  상품 정보
</ResponseField>

<ResponseField name="selectedOptions" type="SelectedOptionRequest[]" required>
  선택된 옵션 정보
</ResponseField>

### ProductRequest

<ResponseField name="sellerId" type="number">
  판매자 ID (선택사항)
</ResponseField>

<ResponseField name="title" type="string" required>
  상품 제목
</ResponseField>

<ResponseField name="price" type="number" required>
  상품 가격
</ResponseField>

<ResponseField name="originUrl" type="string" required>
  상품 원본 URL
</ResponseField>

<ResponseField name="images" type="string[]" required>
  상품 이미지 URL 배열
</ResponseField>

<ResponseField name="options" type="OptionRequest[]" required>
  상품 옵션 배열
</ResponseField>

### OptionRequest

<ResponseField name="title" type="string" required>
  옵션 제목
</ResponseField>

<ResponseField name="price" type="number" required>
  옵션 가격
</ResponseField>

### SelectedOptionRequest

<ResponseField name="title" type="string" required>
  선택된 옵션 제목
</ResponseField>

<ResponseField name="quantity" type="number" required>
  선택된 옵션 수량
</ResponseField>
